# Contributing

Dieses Repository ist aktuell ein privates Projekt.

## Workflow

- Kleine, klar abgegrenzte Pull Requests erstellen.
- Commit-Messages praezise und fachlich formulieren.
- Aenderungen an Scope oder Architektur im PR kurz begruenden.

## Entwicklungsfokus

- React + PWA
- Funktionell moeglichst nah an `spec/original/src`
- Backend-Annahmen im Frontend nicht hart kodieren

## Qualitaet

- Keine unnoetigen Dateien oder Build-Artefakte committen.
- Dokumentation bei relevanten Architekturentscheidungen aktualisieren.
